flying box js

A little simple, very simple (almost non-existent) javascript 'module', just to add floating info boxes, cause why use title="" when you can have a box that appears on hover and follow the mouse movement, that is much cooler.

installation:

file.js

example usage:

most simple:

<div data-flying_box
     data-flbx_text="Some info!"> Element
</div>

result:

Element

other optional params:

<div data-flying_box="1" <!-- same as just 'flying_box' -->
     data-flbx_text="Some other info!"
     data-flbx_move_trans_ms="0" <!-- add transition in milliseconds -->
     data-flbx_offset_x="140" <!-- add X offset (horizontal) -->
     data-flbx_offset_y="0" <!-- add Y offset (vertical) -->
     data-flbx_fontfamily="sans-serif"
     data-flbx_auto_adj="true" <!-- auto offset (default=true) -->
     data-flbx_remove_std_style="false" <!-- when true will remove all the box style, so its possible to fully custom the element (the box class is "flbx-1" or "flbx-2") -->
     data-flbx_box_id="bx1" <!-- add id to the specific box -->
> Element
</div>

result:

Element

other box style:

<div data-flying_box="2" <!-- other box style (2). just a static horizontally centered box. The element need to have a position style (relative, absolute..) -->
     data-flbx_text="A warning!"> Element
</div>

Element

</div>
